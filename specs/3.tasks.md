# Development Tasks

This document outlines the development tasks for the `flood-tide` library, based on the design specified in `specs/2.design.md`.

## Core Components

+ [x] **Task 1: Implement Core Data Structures**
  - [x] Define the `Opt` struct.
  - [x] Define the `NameVal` struct.
  - [x] Define the `Tokens` struct.
  - [x] Define the `Arg` enum.

+ [x] **Task 2: Implement Error Handling**
  - [x] Define the `OptParseErrorKind` enum.
  - [x] Define the `OptParseError` struct and its methods.
  - [x] Define the `OptParseErrors` struct and its methods (for when `single_error` is not enabled).

+ [x] **Task 3: Implement the Lexer (`Lex` struct)**
  - [x] Implement the `Lex::create_with` constructor.

+ [x] **Task 4: Implement Long Option Parsing**

  - [x] Handle `--option value` syntax.

+ [x] **Task 5: Implement Short Option Parsing**
  - [x] Implement the `parse_short_name` method.

+ [x] **Task 10: Implement `abbreviate` Flag**
